# Gestor de Tareas - Modulo 6

Este proyecto es una aplicacion web para la gestion de tareas. Permite a los usuarios organizar, visualizar y administrar sus actividades diarias.

## Caracteristicas Principales

* Interfaz de usuario intuitiva.
* Visualizacion clara de las tareas.
* Gestion del estado de las tareas.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Instalacion y Uso

1. Clona este repositorio en tu maquina local.
2. Abre la carpeta del proyecto.
3. Ejecuta el archivo index.html en tu navegador web preferido para empezar a utilizar la aplicacion.
